# Code of Conduct

This project expects respectful, constructive collaboration.

## Expected Behavior

- Be respectful in language and tone.
- Assume good intent, but be precise when giving feedback.
- Keep technical disagreements focused on code, design, and behavior.
- Help keep bug reports and discussions actionable.

## Unacceptable Behavior

- Harassment, hate speech, or personal attacks
- Doxxing or sharing private information
- Abusive, insulting, or intentionally disruptive behavior
- Repeated bad-faith arguments after maintainers ask to stop

## Maintainer Responsibilities

Maintainers may edit, hide, lock, or remove content that violates this code of conduct. Repeated or serious violations may result in a temporary or permanent ban from participation.

## Reporting

If something needs moderator attention, open a private channel request through the repository support flow described in [SUPPORT.md](SUPPORT.md). Do not escalate personal conflicts in unrelated issues or pull requests.
