# Support

Use the right channel so maintainers can respond efficiently.

## Questions And Setup Help

Use GitHub Discussions for usage questions, setup help, and rule design advice.

## Bug Reports

Use GitHub Issues for reproducible defects. Include Android version, app version, steps to reproduce, and logs or screenshots when relevant.

## Security Issues

Follow [SECURITY.md](SECURITY.md). Do not post sensitive exploit details in a public issue.

## Before Opening A Ticket

- Check the latest release
- Read the [Usage Guide](docs/USAGE.md)
- Search existing issues and discussions first
